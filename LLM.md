# LLM Summary

## Project Overview

Personal portfolio website for Abid Aboobaker, built as a single-page Astro site and deployed to Cloudflare Pages.

## Tech Stack

- Astro 5
- TypeScript (strict)
- Tailwind CSS with custom theme and animations

## Key Files

- `src/pages/index.astro`: Main page content and interactive elements
- `src/layouts/Layout.astro`: SEO metadata, layout structure
- `tailwind.config.mjs`: Custom colors, fonts, keyframe animations
- `astro.config.mjs`: Site configuration and integrations

## Common Commands

- `npm install`
- `npm run dev`
- `npm run build`
- `astro check`
