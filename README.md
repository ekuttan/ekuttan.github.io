# ekuttan.github.io

Personal portfolio website for Abid Aboobaker.

## Development

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build

# Preview build
npm run preview
```

## Tech Stack

- Astro 3.4
- Tailwind CSS
- TypeScript

## Deployment

Automatically deployed to GitHub Pages via GitHub Actions.

Site: [ekuttan.in](https://ekuttan.in)
